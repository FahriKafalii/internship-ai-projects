# Regression Final Project

An end-to-end regression project covering data preprocessing, feature engineering, model training, and evaluation.

## What it does
- Cleans and preprocesses tabular data
- Builds regression models
- Evaluates performance using standard metrics

## Tech stack
- Python
- Pandas, NumPy
- Scikit-learn
- Jupyter Notebook

## How to run
1. Create a virtual environment
2. Install required libraries
3. Open and run the notebook(s) in this folder

## Notes
- Datasets are excluded due to size constraints.
- This project demonstrates a complete ML pipeline.
