# SağlıkGPT

A domain-specific GPT-style application focused on healthcare-related questions with controlled and ethical response generation.

## What it does
- Answers health-related queries using prompt engineering
- Applies guardrails to reduce unsafe or misleading outputs
- Designed for informational purposes (not medical diagnosis)

## Tech stack
- Python
- Prompt engineering
- LLM-based text generation (API keys not included)

## How to run
1. Create a virtual environment
2. Install dependencies (see project files)
3. Create `.env` from `.env.example` if required
4. Run the main application script

## Notes
- API keys and sensitive files are intentionally excluded.
- This project is for educational and demonstration purposes only.
